# 启动选项

![image](./app/src/main/res/mipmap/ic_launcher.png)

## 概要
本Android应用程序是一个电源菜单应用。

使用本应用可以降低使用电源键的频率从而延长其寿命。

请查看[这里](./app/src/main/res/raw-zh-rCN/help.md)获得更详细的帮助信息。
